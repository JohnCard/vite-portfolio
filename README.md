# Vite Portfolio

This is an online portfolio designed to showcase technical skills as a Junior Full-Stack Web Developer, it serves as a dynamic frontend client that interacts with an external backend server. The application is built with React to create reusable and interactive components with configurable props, and Material UI to provide a responsive and visually appealing user interface, the project leverages JavaScript for dynamic behavior.
It was developed to interact with a backend server built using Django REST Framework. For this reason, it was deployed on Vercel, where the frontend application is cloned from GitHub, its dependencies are installed, and the application is built and served to communicate with the backend server.

**Demonstration video about how to use this server:** https://youtu.be/_mh8EzlQFT8

## Backend

External backend powered by **Django REST Framework**.

link for backend server repository - https://github.com/JohnCard/Backend---codeandote

## Features

Modern and Responsive UI: Designed to work across various devices

REST API Integration: Seamless communication with the backend server

Modular Architecture: Built for scalability and easy maintenance

User-Friendly Navigation: Optimized for a smooth user experience

## Technologies

Frontend: React, Vite, JavaScript (ES6+)

UI Framework: Material UI (MUI)

Data Fetching: Axios for REST API integration

Backend Integration: External API powered by Django REST Framework

## Installation

```bash
git clone https://github.com/JohnCard/vite-portfolio.git
cd vite-portfolio
npm install
npm run dev
```

## Deployment

The frontend is deployed on Vercel. The workflow involves cloning the repository from GitHub, installing dependencies, and building the application to communicate with the Django-based backend server
